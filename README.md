# AskJenneAI 🤖

🚀 AI-powered search assistant inspired by Perplexity AI.
AskJenneAI provides real-time answers with source citations using web search.

---

## System Architecture
AskJenneAI follows a production-style, RAG-inspired workflow where user queries are processed, relevant web sources are retrieved, filtered, and transformed into source-grounded answers with evaluation signals such as confidence and hallucination risk.

---

## 🚀 Features
- 🔍 AI search chatbot
- 🌐 Real-time web results
- 📚 Source citations
- 💬 ChatGPT-style interface
- ⚡ Fast response system
- 🛠️ Error handling

---
